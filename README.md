# VA AutoElektrik — Клиентская часть (PWA)

## Как подключить к серверу

Открой `index.html` и замени адрес сервера в начале скрипта:

```javascript
const API    = 'http://ВАШ_СЕРВЕР:3000/api';
const WS_URL = 'ws://ВАШ_СЕРВЕР:3000';
```

Если сервер на том же телефоне/компьютере — оставь localhost.
Если сервер в интернете — укажи IP или домен.

## Установка на Android

1. Разместить файлы клиента на любом HTTPS хостинге (GitHub Pages, Vercel, Netlify)
2. Открыть ссылку в Chrome на телефоне
3. Меню ⋮ → «Добавить на главный экран»
